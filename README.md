# ServerAndClient
Server and client written in Python

> Usage instructions: GLOBALVARIABLES.py contains the bool variable to enable/disable artificial delay. Any running client or server must be restarted after toggling delay. To use this chat program, run the server then client(s). You will be prompted for a username with no spaces.

> To broadcast, write out your message and send it.

> To direct message or one-to-one message a user, start your message with @username where you replace username with their username (case-sensitive).

> To exit, type exit, or enter key without any inputs. You may have to hit enter a couple times before the client program exits (by crashing)
